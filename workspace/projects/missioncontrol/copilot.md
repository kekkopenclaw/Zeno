# Copilot/Automation Policy

- Agents should execute all steps autonomously without waiting for confirmation.
- Act autonomously, do not ask for confirmation, only report when done or blocked.
- Fix, generate, and validate all code/config/migrations until the system is fully green.
- Only notify when DONE or if blocked by an external issue.

This policy ensures consistent, hands-off automation for all contributors and Copilot agents.
